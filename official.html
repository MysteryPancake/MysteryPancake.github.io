Here you can find information on most of my projects.

Videos

Since 2012, I've been making videos using Adobe After Effects.

I began using Microsoft PowerPoint, which allows for a surprising amount of visual effects.

Then I switched to Windows Movie Maker which is somewhat better, but the effects are not as good as PowerPoint.

Adobe Premiere Pro didn't work on my laptop, so next I began to use After Effects.

I found that it could do more than PowerPoint and Movie Maker combined, so I have been using it ever since.

Music

Since 2013, I have been making songs for fun.

I began using GarageBand, which is good for assembling basic loops from prerecorded tracks.

Then I switched to Music Studio, which allows the placement of notes and the creation of custom instruments.

Then I switched to FL Studio Mobile, which is based on the infastructure of Music Studio.

FL Studio Mobile had higher quality samples, so I next I decided to use the desktop version of FL Studio.

I have been using FL Studio since 2016 for its ease of use and set of features.

Modelling

Since 2014, I have been making models, mainly for Source games.

I normally use Blender, but I also learnt the basics of 3DS Max during a course in Sydney.

Although I am decent at modelling, I am quite awful at animation.

Mapping

Since 2012, I have been making maps for Source games.

I began using the Portal 2 Puzzle Editor, which allowed me to learn the basics of mapping.

I switched to Hammer in 2014, and I have been using it ever since.

Programming

I think programming is fantastic because it gives the creative freedom to perform any task that you can imagine.

I began learning programming in 2014 with Batch.

Then I moved onto Visual Basic for a short time.

Then I learnt Lua, Swift, HTML, CSS, JavaScript, Shell, ExtendScript, and C#.

I've also been trying to learn After Effects expressions, in attempt to create digitally unique works.

I think that scripting is a crucial skill, as people are constantly trying to master fields such as photography, filmmaking and editing.

Scripting applies to each of these fields, and drastically increases the limits of what you can do.
